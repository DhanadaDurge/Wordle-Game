# Wordle-Game
The Wordle Game is an interactive web-based word guessing game developed using HTML, CSS, and JavaScript. The objective is to guess a hidden five-letter word within a limited number of attempts.

After each guess, the game provides real-time feedback using color indicators:

🟩 Correct letter in the correct position
🟨 Correct letter in the wrong position
⬜ Incorrect letter

This project focuses on creating an engaging user experience with a clean and responsive interface. It demonstrates key front-end concepts such as DOM manipulation, event handling, and logical problem-solving.
